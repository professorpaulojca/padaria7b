# Cascade Memories (curto e objetivo)

Use este arquivo para registrar **padrões que se repetem** e decisões duráveis, por exemplo:
- padrões de erro da API
- conventions de validação
- pitfalls em procedures Postgresql
- decisões sobre compatibilidade

Formato sugerido:
- Data
- Contexto
- Decisão/Insight
- Evidência (arquivo/procedure/endpoint)
- Impacto
