# Regras do Memory Bank (Alta Acurácia)

- Sempre verificar a estrutura do Memory Bank ao iniciar uma sessão.
- `activeContext.md` e `progress.md` são obrigatórios e devem refletir o estado real.
- Decisões técnicas e contratos relevantes devem ser registrados em `systemPatterns.md` ou `contracts.md` (se existir).
- Qualquer hipótese deve ser marcada como **ASSUMPTION** e ligada a uma **DÚVIDA** quando não houver evidência.
