# Project Brief (Brownfield)

## Context
Projeto em andamento (brownfield) com Angular + .NET Core 3.1 + Oracle/PLSQL.
Objetivo principal: estabilização, correção de falhas e alinhamento rigoroso aos requisitos existentes.

## Goals
- Reduzir falhas funcionais e regressões
- Garantir aderência total aos requisitos do cliente
- Estabelecer SPEC como fonte de verdade
- Melhorar rastreabilidade e governança

## Non-Goals
- Reescrita completa do sistema
- Mudanças sem rastreio em SPEC/TASKS
