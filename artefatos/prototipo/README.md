# Protótipos do Sistema Padaria

Este diretório reúne os 13 protótipos HTML do sistema.

## Ponto de entrada

Abra primeiro:

- `index.html` para usar o portal dos protótipos

Atalhos principais:

- `login-dashboard.html` para visão executiva inicial
- `relatorios.html` para consolidação gerencial
- `configuracoes.html` para parâmetros do sistema

## Mapa dos arquivos

- `login-dashboard.html`: login por perfil e dashboard inicial
- `atendimento.html`: pedidos, mesas, cozinha e delivery
- `caixa.html`: abertura, fechamento e pagamentos
- `financeiro.html`: contas, fiado, boletos e DRE
- `cadastros.html`: produtos, clientes, fornecedores e equipe
- `estoque.html`: saldos, lotes, validade e inventário
- `compras.html`: pedidos de compra e recebimento
- `producao.html`: fichas técnicas e ordens de produção
- `pessoal.html`: ponto, escalas, vales e ocorrências
- `higiene.html`: checklists, temperaturas e inspeções
- `manutencao.html`: equipamentos, preventivas e corretivas
- `relatorios.html`: catálogo de relatórios e exportações
- `configuracoes.html`: dados da empresa e parâmetros

## Padrão adotado

- HTML self-contained
- CSS e JS embutidos
- mock data local
- sidebar e header sticky
- tabs internas
- modais e toasts
- responsividade

## Observações

- Os arquivos foram pensados como protótipos navegáveis, não como implementação final integrada.
- A navegação entre módulos acontece por links locais entre os HTMLs.
- Alguns terminais PowerShell podem exibir caracteres com codificação quebrada, mas os arquivos estão em UTF-8.
